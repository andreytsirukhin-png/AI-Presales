# Product scope

## Primary users

- Delivery Manager
- Presales Manager
- Business Analyst
- Solution Architect
- Account Manager

## Core user story

As a presales or delivery professional, I want to upload an RFP and receive a traceable analysis so that I can prepare clarification questions, identify delivery risks and build a more reliable solution estimate.

## MVP capabilities

### In scope

- File upload
- Text extraction
- Document classification
- Structured requirements extraction
- Gap and ambiguity detection
- Clarification-question generation
- Risk generation
- Assumption generation
- Source evidence for every important finding
- Human review
- JSON export

### Out of scope for MVP

- Automatic final pricing
- Fully autonomous solution architecture
- Jira, SharePoint or CRM integration
- Multi-tenant authentication
- Production-grade role management
- Automatic contract generation
- Unsupervised customer communication

## Success metrics

- At least 90% of generated findings include valid source evidence.
- At least 80% of mandatory requirements identified in a manually labeled test set.
- At least 70% of generated questions rated useful by a presales reviewer.
- Zero fabricated page references.
- Analysis of a 100-page document completes without manual preprocessing.
